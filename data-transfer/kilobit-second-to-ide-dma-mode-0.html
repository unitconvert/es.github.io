How to Convert Kilobit/second to IDE (DMA Mode 0)

1 kb/s = 3.04762E-5 IDE (DMA mode 0)
1 IDE (DMA mode 0) = 32812.5 kb/s

Example: convert 15 kb/s to IDE (DMA mode 0):
15 kb/s = 15 × 3.04762E-5 IDE (DMA mode 0) = 0.0004571429 IDE (DMA mode 0)

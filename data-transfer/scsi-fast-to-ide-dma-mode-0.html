How to Convert SCSI (Fast) to IDE (DMA Mode 0)

1 SCSI (Fast) = 2.380952381 IDE (DMA mode 0)
1 IDE (DMA mode 0) = 0.42 SCSI (Fast)

Example: convert 15 SCSI (Fast) to IDE (DMA mode 0):
15 SCSI (Fast) = 15 × 2.380952381 IDE (DMA mode 0) = 35.7142857143 IDE (DMA mode 0)
